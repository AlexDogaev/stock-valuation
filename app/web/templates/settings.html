Настройки и персональная калибровка

Доходность реальна только над ОЩУЩАЕМОЙ инфляцией. Таргет и режим задают порог троичного сигнала. Инфляцию и таргет можно править прямо в шапке.

Таргет и инфляция

Калькулятор таргета цель портфеля % ÷ доля атаки % → hurdle атаки — Защита ≈ 0% реальных → вся доходность с атаки. Уйдёшь в защиту (доля атаки ↓) — планка растёт сама = меньше аппетит.

Сигнал ЦБ по ставке — ручной ввод (override авто-фетча)

Opus градирует траекторию КС по пейсу решений + риторике. Авто-фетч тянет заявление с cbr.ru/press/keypr. Если авто сломался или хочешь задать конкретный текст — вставь сюда (пусто → снова авто). Нужен ключ Opus, иначе градация идёт по числовому пейсу.

Фундаментал уровня 2 — T-Invest API

Обновить ROE, прибыль, payout, капитал по эмитентам из T-Invest (GetAssetFundamentals). Нужен токен: env TINVEST_TOKEN или файл .tinvest_token в корне проекта. Дополняет seed из Excel.
