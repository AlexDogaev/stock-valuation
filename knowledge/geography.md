# География РФ на 20 лет: «две России» (тектоника ур.0, пространственный слой)

Свёртка из NOTES_методология §7 (геоветка). Не инвестсовет — структурное знание для рамы.
Главное правило: **рынок РФ считать ПО УЗЛАМ, не по карте.**

## Великий Аттрактор — концентрация ДЕФИЦИТА, не изобилия
Живое население стягивается в мегаагломерации. НО (анти-китайская оговорка): Китай концентрировал РОСТ
(живая многолюдная периферия), РФ концентрирует УБЫЛЬ — узел растёт за счёт обескровливания вымирающей
периферии. Геометрия ближе к Канаде/Австралии (узкая обитаемая полоса, пустой центр), но те растут на
миграции, РФ депопулирует. **Концентрация ≠ рост:** агломерация = ПЕРЕРАСПРЕДЕЛЕНИЕ сжимающегося пирога,
игра около нуля (с общей убылью — отрицательная). НЕ экстраполировать китайский агломерационный рост.
Аттрактор затухает (резерв села исчерпан, пик ранних пятилеток) + госинфраструктура (ВСМ/БАМ/автобаны)
СЛЕДУЕТ за концентрацией и усиливает её (периферию обслуживать нерентабельно).

## Четыре типа судьбы территорий
1. **ГОРЛЫШКИ живой экономики** (стягивают): Москва ~22 млн (ПЛАТО, пик ~2035, откат к 2045 → «вечный рост
   на притоке» НЕ держится), Питер ~7 млн (вялый рост→плато), Юг-Краснодар ~6-7 млн (единственный ОРГАНИЧЕСКИЙ
   рост — климат; но 5 пределов: мелеющий донор, инфра, вода/засуха, качество роста, геополитика).
2. **ОСОБЫЙ случай:** Сев.Кавказ (Чечня/Дагестан) — рост на РОЖДАЕМОСТИ (свой мотор, не миграция).
3. **ОЧАГИ за Уралом** (держатся): нефтегаз Зап.Сибири (вахта+города), южносибирские миллионники
   (Новосибирск/Красноярск), якоря ДВ (Владивосток/Хабаровск/Якутск/Благовещенск).
4. **ЗОНА УБЫЛИ/ПРОБОЯ:** европ.периферия + ~800-900 малых городов + Вост.Сибирь + ДВ вне якорей.

Механизм: Москва+Питер+Юг тянут из ОДНОГО мелеющего резервуара (внутр.миграция из убывающих регионов) →
упрутся в его дно ВМЕСТЕ. Малые города: юридически не исчезают (даже растут числом — усыхают ниже порога
статуса), но ЖИВЫХ резко меньше — «города-призраки». ДВ поляризуется (архипелаг якорей в растущей пустоте),
7,8→~6,5-7 млн к 2045; ресурсно-транзитный контур (вахта+порты), не потребрынок.

## Логистика — СКВОЗНОЙ фактор (специфика РФ: масштаб территории)
Не отдельная колонка, а множитель в трёх ролях:
- **РОВ** — главный недооценённый ров маркетплейсов/ритейла (X5/Озон/WB): логистсеть в стране размером с РФ =
  непробиваемый барьер входа (капитал+время+плотность). Физический ров, ИИ-волной НЕ смывается (контраст с tech).
  Конкретное имя барьера «новый мега-игрок не родится».
- **НАЛОГ** — «тирания расстояния» давит производительность → часть низкого базового g. Водитель = непокрываемый
  ИИ физтруд, дефицитный, дорожает («сварщик на колёсах»). Госстройка (коридоры между центрами) последнюю милю
  в глубинке НЕ снимает.
- **АПСАЙД** — автоматизация (автономная МАГИСТРАЛЬ для РФ-расстояний реалистичнее роботакси), но ур.1, гадательный
  тайминг → не в базу. Главный неиспользованный рычаг g.
Отдельный контур — чистые логистимена (порты/НМТП, Совкомфлот, восточный полигон) с геополитическим профилем.

## Экосистемы и квази-деньги
По весу — ДУОПОЛИЯ Сбер+Яндекс (VK/МТС/Т-Банк кратно меньше; Т-Банк рос кратно, финансовое ядро → веса могут
перетасоваться на 20 лет). Lock-in валюты (Спасибо/Плюс/кешбэк) — реальный ров, НО квази-деньги = инструмент
lock-in, НЕ параллельная валюта: потолок ниже, сверху давят (1) насыщение подписок (прирост замедлился) и
(2) ЦИФРОВОЙ РУБЛЬ (конкуренция за контроль транзакций/данных; госнавес не любит частные контуры стоимости).
Структурное напряжение: частный lock-in/данные ↔ госконтроль. Озон — транзакционный маркетплейс+финтех, НЕ
лайфстайл-экосистема с квази-валютой уровня Плюса.

## Продовольствие (уточнение по X5)
Зрелый рынок с СИЛЬНЕЙШИМ рвом (логистика/плотность/масштаб непробиваемы), НЕ зона дизрупции (контраст с tech).
Динамика = КОНСОЛИДАЦИЯ лидером + расширение TAM (прод.е-ком + АПТЕКИ: рец-эксперимент Москва/МО/Белгород
завершается 2026, при успехе вся РФ; аптеки маржа ~1% беззащитны → X5/маркетплейсы консолидируют). →
продовольствие УСИЛИВАЕТ X5 (зрелый ров + расширение TAM), не «сдвиги/риск».

## Логистика-линза: правило позиции в цепочке (NOTES финал)
Маржу захватывает тот, кто контролирует УЗКОЕ МЕСТО / последнюю милю, не «логистика вообще».
Ось спроса: слой исполнителей расщепляется (магистраль/последняя миля/фулфилмент — разная защищённость).
C2C захвачен маркетплейсами — водораздел по ФОРМАТУ ГРУЗА (мелкий штучный vs паллет/FTL), не по типу
клиента (B2B/B2C). Космос — enabler-фон, НЕ инвестируемая ставка для РФ-портфеля.

## Курс рубля — трансмиссия и секторная асимметрия (макро ур.2)
Ослабление рубля: экспортёрам выручка↑ (девал-хедж — уже в теге EXPORTER/currency_profile + сниженном
макро-штрафе hurdle), внутренним — издержки импорта/оборудования↑ при рублёвой выручке (встречный).
Курс теперь двояк: и ФАКТОР оценки эмитента (маршрутизация тектоники/штрафа по DOMESTIC/EXPORTER), и
самостоятельный ИНСТРУМЕНТ в меню (FX-модуль). Распределение курса на горизонт (сценарии) — вход FX/оценки.

## Мультиассет (carry/bonds/fx/PD) — логика внедрена, данные следующей фазой
Реализованы ЧИСТЫЕ модули: `carry` (единый hurdle по траектории КС), `bonds` (троичный: real_YTM/spread/
rate_signal/credit_ok), `credit_pd` (4-слойный синтез PD с конвергенцией), `fx` (E[курс]+купон−carry,
правило доминирования, левый хвост). Данные (MOEX ISS бонды + КБД ЦБ + рейтинги→PD) и UI-меню сравнения
классов — СЛЕДУЮЩАЯ фаза (живой API + курируемая бонд-вселенная).

## Инвест-итог (всё → одно правило)
Реальный РАСТУЩИЙ потребрынок = Москва(плато)+Питер+Юг+молодой Кавказ; остальное стягивается в них либо
ресурсно-вахтовый контур (сырьё, не потребитель). Внутренние потреб/фин имена портфеля (X5/Сбер/Т/Озон/MDMG)
живут ЦЕЛИКОМ в «живой России узлов». Считать рынок по узлам, не по карте: ~84% городов выпадают из живой
экономики, оставаясь на ней юридически. Консолидатор берёт долю в стягивающемся (не растущем) пироге.
Московская недвижимость устойчива на притоке (но выход на плато ~2035), периферийная обесценивается.
