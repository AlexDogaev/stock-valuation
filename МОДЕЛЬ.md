# Модель оценки активов MOEX — полное описание

> Личный мульти-ассет сервис оценки справедливой стоимости активов Московской биржи: **акции + облигации + валюта** под единой рамкой. Прод: `https://invest.solarwings.tech`. Стек: FastAPI + SQLite + Jinja2 + ванильный JS; аналитика — Claude Opus.
>
> Документ описывает методологию (что и как считается), а не код построчно. Актуально на 20.06.2026.

---

## 0. Философия и базовые принципы

1. **Троичный сигнал, а не точечная цена.** Везде выход — `ПОКУПАЙ / ГРАНИЦА / ВОЗДЕРЖИСЬ` с буфером (маржой безопасности). Точность входов не критична: сигнал робастен к ошибке 10–20%, критична **верифицируемость и консистентность определений**.
2. **Реальная доходность над инфляцией.** Всё считается в реальных величинах (после инфляции). Номинальные цифры обманывают при высокой инфляции РФ.
3. **Машина считает считаемое, человек судит о несчитаемом.** Количественный слой — формулы; качественный (структурный, тектонический) — экспертное суждение, корректирующее рост.
4. **Один фактор учитывается один раз.** Жёсткий анти-двойной-счёт (госнавес, демография, соинвест, шок-инфляция).
5. **Подчинение единой макро-модели.** Все классы активов считаются от одного макро-прогноза (инфляция + риск шока) и единого carry-ориентира.

## 2. Верхний слой: Макро-прогноз (`macro_outlook.py`)

Первичный объект модели. От него считаются ВСЕ доходности. Два фундамента: **прогноз инфляции** и **риск шока**.

### 2.1. Прогноз инфляции — срочная структура

Инфляция не одна цифра, а **кривая по сроку**: высокая сейчас (`felt`, вводится вручную, ~14%) → нормализуется к терминалу (`terminal`, из траектории КС) за окно дезинфляции, потом плоско.

**Средняя инфляция до погашения бумаги срока M** (`inflation_to_maturity`):
```
инфляция глайдит felt → terminal за norm_years (окно), затем = terminal
E[инфл|M] = геом. среднее этой кривой на [0, M]
```
- Короткая бумага видит в основном **высокую текущую** инфляцию → её реальная доходность ниже.
- Длинная — в основном **терминал** → реальная выше.
- Это деривация реальной доходности к погашению: `real_YTM(M) = nominal_YTM(M) − E[инфл|M]`.

**Окно дезинфляции `norm_years` — НЕ хардкод**, выводится по приоритету:
1. Оценка Opus из **риторики ЦБ** (`disinflation_months` — за сколько месяцев инфляция выйдет на терминал).
2. Расчёт по **темпу траектории**: `|КС − терминал|пп / |средний шаг за заседание| / 8 заседаний-в-год` (границы 0.5–4 года).
3. Резерв 2 года, если траектории нет.

### 2.2. Риск шока — дерево «база + шок» с вектором

Инфляция и шок **не ортогональны** (шок сам разгоняет инфляцию), поэтому моделируются **деревом из двух веток**, а не «база минус штраф»:

| Ветка | Вероятность | Что происходит |
|---|---|---|
| **База** | 1 − p_shock | дезинфляция к терминалу, нормальный рост |
| **Шок** | p_shock (годовой hazard) | инфляция↑, курс↓, КС↑, акции↓ |

**Шок описан вектором по факторам** (калибровка по 2014 и 2022, уточняется Opus):

| Параметр | Дефолт (2014/2022) | Смысл |
|---|---|---|
| `equity_dd` | −37% | **условная** просадка IMOEX *если* шок (не безусловная) |
| `recovery_1y` | 55% | доля просадки, отыгранная за год (хвост восстановления) |
| `fx_pct` | +40% | девальвация рубля |
| `ks_pp` | +8.5пп | реакция ЦБ (2014→17%, 2022→20%) |
| `infl_pp` | +9пп | всплеск инфляции |
| `sectoral` | экспортёры −18 / внутренние −45 / банки −45 / IT −50 | секторальные просадки (девальвация смягчает экспортёров) |

Годовой hazard `p` — оценка Opus (`aggregate_pct`). Дефолт по режиму, если нет: NORMAL 10% / RISK 25% / SHOCK 45%.

### 2.3. Агрегаты прогноза

- **E[инфляция]** = база + стационарный вклад шок-инфляции:
  ```
  E[инфл|M] = базовая_инфляция(M) + infl_pp · min(1, hazard · norm_years)
  ```
  `min(1, hazard·norm_years)` = доля «шоковых» лет (частота × длительность). Учитывает ~hazard·H шоков за длинный горизонт (а не один). Срочная вариация реала идёт через БАЗУ, шок-инфляция per-year постоянна.

- **Курс-сценарии для FX** = две ветки дерева:
  ```
  [(1−p, базовый дрейф), (p, девальвация в шоке)]
  базовый дрейф ≈ terminal − мировая инфляция 3%
  ```

- **Carry** (`carry.py`) — единый рублёвый безриск = средняя КС по траектории за горизонт (линейный глайд current→terminal). Альтернативная стоимость капитала для всех классов («припарковать в ОФЗ»).

## 3. Облигации (`bonds.py`, `credit_pd.py`, `engine.screen_bonds`)

Защитный рукав портфеля. Бар = бить инфляцию (real ≥ 0) + кредит, **не** +10% таргет атаки акций.

### 3.1. ОФЗ-кривая и спред

- **Кривая КБД** строится из самих ОФЗ-ФИКС (дюрация → YTM), без cbr.ru (там cert-проблемы). Линейная интерполяция на дюрацию.
- **Спред корпората** = `YTM − КБД(дюрация)`.
- Оферта → **yield-to-put** (не -to-maturity) — критично для РФ.

### 3.2. Тип купона (из MOEX `BONDTYPE`)

| Тип | Что | Ставочный сигнал |
|---|---|---|
| **Фикс** | фиксированный купон (ОФЗ-ПД 26xxx) | выигрывает при ↓КС (capital gain ≈ дюрация×Δставки) |
| **Флоат** | плавающий (ОФЗ-ПК 24/29xxx) | выигрывает при ↑КС (купон переоценивается вверх) — **инвертирован** |
| **Линкер** | индексируемый номинал (ОФЗ-ИН 52xxx) | **YTM уже реальный** → инфляцию НЕ вычитаем |

Структурные/конвертируемые/валютные — вне скринера (не классические бонды).

### 3.3. Кредитный риск (PD) — 4 слоя, конвергенция

PD нелинейна по рейтингу (BB- ~12%, B ~27%, CCC ~71% годовых). Четыре независимых источника → синтез:

1. **PD_rating** — матрица дефолтов агентства (запаздывает).
2. **PD_market** — из спреда: `(спред − премия_ликв 0.5%) / LGD`. Опережающий.
3. **PD_fundamental** — скоринг отчётности (Долг/EBITDA, покрытие, refinancing wall).
4. **context_modifier** — отрасль / госнавес / implicit support.

**Синтез:** сходятся → среднее (надёжно); **расходятся >10пп → red flag, берём консервативно max** + требуем больше MoS. `LGD = 65%` (РФ-корпораты).

> Текущий статус: на скринере используется интерим — рыночная PD из спреда с cap 10% (отсекает junk-ВДО), под **стресс-множителем макро-режима** `1 + 0.5·F`. Независимая PD (рейтинги АКРА/ЭкспертРА + фундаментал) — фаза 2b.

### 3.4. Реальная и риск-скорректированная доходность

- `real_YTM = YTM − E[инфл за срок бумаги]` (линкер: = YTM).
- **Дефолт-риск 12м** = годовая PD (под стресс-множителем режима).
- **Дефолт-риск за срок** (кумулятив) = `1 − (1 − PD_год)^дюрация` — растёт с дюрацией.
- **Реал.−риск** (risk-adjusted) = `real_YTM − PD_год × LGD` (за вычетом ожидаемых годовых кредитных потерь). Скринер сортируется по ней.

### 3.5. Сигнал по облигации
```
ПОКУПАЙ  если real_YTM ≥ hurdle + buffer И ставочный сигнал не встречный И credit_ok
ВОЗДЕРЖИСЬ если real_YTM < hurdle − buffer ИЛИ credit_ok = false
ГРАНИЦА  иначе
```
Для бондов: `hurdle = 0` (защита, бар = бить инфляцию), `buffer = 1%`.

## 4. Валюта / FX (`fx.py`, `engine.screen_fx`)

Закрытие акцие-центричной слепоты: на горизонте валюта может быть лучшим вложением. Инструменты — **замещайки/юаневые бонды** (MOEX, FACEUNIT ≠ RUB).

```
E[отдача, ₽] = FX-YTM + E[курс по распределению] − carry
```
- `E[курс]` — из курс-сценариев макро-прогноза (база-дрейф + ветка девальвации в шоке), **не хардкод**.
- Сравнение с баром (бить рублёвый carry) при **повышенном MoS** (курс ненадёжен → 2× буфер).
- **Правило доминирования:** при наличии купонного аналога голую валюту не предлагать (тот же курсовой риск + купон). Замещайка — купонный инструмент, не доминируема.
- Левый хвост (укрепление рубля) учитывается в риске, не только в EV.

## 5. Акции (`engine.evaluate_issuer` + `valuation.py` + `structural.py` + `tectonic.py`)

Самый сложный класс. Полный прогон эмитента комбинирует количественный, структурный, тектонический и макро-слои.

### 5.1. Количественный слой (`valuation.py`)

**Главная формула полной доходности** (зрелый/растущий режим):
```
g_итог       = g_базовый × структурный_множитель + тектоническая_поправка
полная_ном   = (1 + g_итог) · сжатие_мультипликатора − 1 + дивдоходность
реальная     = (1 + полная_ном) / (1 + дефлятор) − 1     [Фишер]
сигнал       = троичный(реальная, hurdle, buffer)
```
- **Сжатие мультипликатора** обязательно (зрелые = 1, растущие < 1) — рынок переплачивает за рост, мультипликатор сожмётся.
- **Устойчивый рост** `g = ROE × (1 − payout)` — не задаётся произвольно.

**Justified-мультипликаторы** (зрелый режим):
```
справедл. P/B = (ROE − g) / (r − g)
справедл. P/E = payout·(1 + g) / (r − g)
справедл. капа = Капитал × P/B
```

**Тест достоверности (зона r−g):** цена взрывается при r→g. Спред `r − g`: ≥2.5пп «применимо», 1.5–2.5пп «хрупко», <1.5пп «вне зоны» (точная цифра не выдаётся).

**Тест «аванс в цене»** (оптимизм priced-in): `имплицир. прибыль = капа / нормальный_P/E (6.0)`. Если кратно (≥2×) выше текущей — рынок заложил рост авансом, апсайд только при превышении → флаг.

### 5.2. Структурный слой (`structural.py`) — качественное суждение

Балл по 4 измерениям (−2 угроза … +2 попутно): **ров + дизрупция + секулярный TAM + регуляторика**.
```
балл ≤ −4  → множитель 0.0  (вырождающийся, g обнуляется)
балл < 0   → 0.5
балл ≥ 4   → 1.1
иначе      → 1.0
```
- Демография в балл **не входит** (поднята на top-down уровень — тектоника, анти-двойной-счёт).
- Госнавес в балл **не входит** (учитывается один раз: либо в низком g, либо отдельно).
- Дизрупция — гадание (низкая уверенность) → как риск-дисконт, не факт.

### 5.3. Тектоническая рама (`tectonic.py`) — арбитраж горизонта

Пятиуровневая иерархия факторов по скорости/отыгранности:
```
ур.0 ТЕКТОНИКА (демография, EROI) → ур.1 институты → ур.2 макро →
ур.3 настроение → ур.4 эмитент
```
Верхние уровни **хуже отыграны** (рынок близорук <3–5 лет под высокую ставку дисконта) → edge = **арбитраж горизонта**.

**Выход:** поправка к `g` эмитента из его **сектора × текущей пятилетки**, коридор −1.5…+3пп (намеренно скромен — тектоника двигает g медленно).
- Базовый g рынка немонотонный: P1(2026-30) +1.5% → P4(2041-45) +0.3%. Реального «прилива» нет → альфа в консолидаторах, не в индексе.
- Демофакторы (калибровка на пирамиде Росстата-2046): старение (ровное плато ~+1.5-1.8пп), рождаемость (дно P1), раб.сила расщеплена по покрываемости ИИ (физтруд дорожает / клерикал дешевеет), агломерация, EROI.
- **Маршрутизация:** РФ-тектоника спроса только для DOMESTIC. EXPORTER → 0 (их спрос по глобальной демографии).

### 5.4. Макро-поправка hurdle (`engine.macro_fragility`, `macro_hurdle_delta`)

Здоровье ФНБ + риск шока → системная риск-премия к требуемой доходности:
```
F (индекс хрупкости ∈[0,1]) = 0.5·(deval_score/6) + 0.5·max(0,(shock_pct−15)/25)
F ≤ 0.30  → бонус (ниже hurdle, агрессивнее)
F > 0.30  → штраф (выше hurdle), но МЕНЬШЕ для качества (барбелл) и экспортёров (хедж девальвации)
```
- Множитель валютной хрупкости: EXPORTER 0.3 / MIXED 0.7 / DOMESTIC 1.0.
- Риск шока ≥ 80% — жёсткий потолок (покупать нет смысла).
- В **реализованном** режиме ШОК hurdle обнуляется (добор подешевевшего качества) — форвардная осторожность уступает.

### 5.5. Прочие слои эмитента
- **Налоговый слой:** дивиденды −НДФЛ ежегодно; курсовой рост — ЛДВ освобождает при горизонте ≥3 лет / ИИС-3. Сигнал на посленалоговой основе (иначе дивидендные имена завышены против ростовых).
- **Маркер качества** (PROVEN_QUALITY / PROSPECTIVE / …) — гейт + вход в макро-поправку.
- **Классификация** (ROIC vs WACC, фаза роста) — режим оценки.
- **Зрелая оценка** — справедливая капа, вердикт (Недооценён/Справедливо/Переоценён ±10%), implied-доходность, нужная просадка под hurdle.
- **Калибровка дивспайка:** сигнал на устойчивой дивдоходности (`payout × прибыль / капа`), не на разовой TTM-выплате.

## 6. Сценарий по горизонтам (`engine.scenario_table`)

«Если вложиться в инструмент на 3/5/10/20 лет и не трогать — какая реальная доходность ждёт». Тайминг шока **равновероятен по месяцам**.

### 6.1. Кумулятивная вероятность шока
```
P(≥1 шок за H лет) = 1 − (1 − hazard)^H
```
При hazard 31%: 3г 67% → 5г 84% → 10г 98% → 20г ~100% (за 20 лет шок почти наверняка — референс).

### 6.2. Реальная доходность по классам

| Класс | Расчёт | Шок |
|---|---|---|
| **ОФЗ-фикс** (HtM) | `YTM − E[инфл|H]` | бьёт только через инфляцию (цена → номинал) |
| **Замещайка** | `FX-YTM + E[курс] − E[инфл]` | девальвация **в плюс** |
| **Акции IMOEX** | `(ОФЗ-реал + премия 4пп) − драг шоков` | просадка через драг |

**Драг шоков для акций** (`equity_shock_drag`) — годовой реальный убыток, тайминг по месяцам:
```
шок в месяце m: перманентная потеря D·(1−r) (компаундится по всем шокам)
                + временный остаток D·r, не отыгранный к концу горизонта (если шок в последний год)
усреднение по всем месяцам = равновероятный тайминг
```
где D = условная просадка, r = доля восстановления за год. Включается от 3 лет.

### 6.3. Годовая vs накопленная
В каждой ячейке: **годовая реальная (CAGR) · всего за срок**:
```
всего за срок = (1 + годовая_реальная)^H − 1
```
Годовая почти плоская (~флэт после окна дезинфляции), но накопленная растёт кратно за счёт компаундинга (ОФЗ +18% за 3г → +306% за 20л). **Ключ:** горизонт работает через сложный процент и рост вероятности поймать кризис, а не через рост годовой ставки.

## 7. Траектория ключевой ставки (`rate_trajectory.py`, `llm_macro`)

Opus классифицирует траекторию КС по темпу решений ЦБ + риторике заявлений (7 грейдов: агрессивное/обычное/медленное снижение, удержание, …повышение). Числовой fallback по среднему шагу за заседание (пороги: <0.13пп удержание, ≥0.35 обычное, ≥0.85 агрессивное).

Выходы, кормящие модель:
- **Терминальная КС** → терминальная инфляция = `терминал − реальный спред 2.5%` → дефлятор и срочная структура инфляции.
- **Направление** (cut/hold/hike) → ставочный сигнал бондов, carry.
- **Окно дезинфляции** (см. 2.1).

## 9. Сводка ключевых формул

```
Реальная (Фишер):        real = (1+ном)/(1+дефл) − 1
Устойчивый рост:         g = ROE·(1 − payout)
Справедл. P/B:           (ROE − g)/(r − g)
Инфляция к сроку:        геом. среднее глайда felt→terminal за окно, потом terminal
E[инфляция]:             база(M) + infl_pp·min(1, hazard·norm_years)
Реал.YTM бонда:          YTM − E[инфл|срок]
PD из спреда:            (спред − 0.5%)/LGD,  LGD=65%
Дефолт за срок:          1 − (1−PD_год)^дюрация
Risk-adj YTM:            real_YTM − PD_год·LGD
Carry:                   средняя КС по глайду current→terminal за горизонт
Кумул. P(шок):           1 − (1−hazard)^H
Накопленная реальная:    (1 + годовая_реальная)^H − 1
Индекс хрупкости F:      0.5·(deval/6) + 0.5·max(0,(shock−15)/25)
Окно дезинфляции:        |КС−терминал|пп / |шаг/заседание| / 8   (или Opus из риторики)
```

## 10. Допущения и калибровки
